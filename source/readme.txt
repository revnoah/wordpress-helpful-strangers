WordPress plugin to several helper functions present in other frameworks to help with development.

== Description ==

WordPress plugin to several helper functions present in other frameworks to help with development. These tools are 
designed to help with the development and publishing of WordPress plugins and themes.

== Features ==

*   Laravel helper functions
*   General functions not related to a specific framework

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/helpful-strangers` directory, or install the plugin through 
the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Helpful Strangers screen to configure the plugin.

== Usage ==

The best way to familiarize yourself with the functions is to review the code.
